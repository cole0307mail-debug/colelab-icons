## Core Capabilities

- **Standardized icon asset management**: one source, one naming system, one theme taxonomy.
- **Traceable generation pipeline**: from raw SVG assets to framework-ready components.
- **Multi-package collaboration**: aligned layering with `icons-svg` and `icons-react`.
- **Stable delivery**: reusable across products with predictable output.

## Package Structure

- `@colelab/icons-svg`: source icon definitions.
- `@colelab/icons-vue`: Vue adapter layer.
- `@colelab/icons-react`: React adapter layer.

## Design Goals

1. **Consistency** across pages and business domains.
2. **Maintainability** through clear rules and structure.
3. **Scalability** for multi-framework output.
4. **Practicality** for real-world team adoption.
